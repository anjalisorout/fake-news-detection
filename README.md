# fake-news-detection# 📰 Fake News Detection (ML Project)

## 📌 Description

This project detects whether a news article is real or fake using machine learning and NLP techniques.

## 🚀 Features

* Text classification
* TF-IDF vectorization
* Logistic Regression

## 🛠️ Technologies

* Python
* scikit-learn
* Pandas

## ▶️ How to Run

python fake_news.py
